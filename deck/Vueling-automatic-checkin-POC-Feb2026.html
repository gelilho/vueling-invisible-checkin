The Project

Automatic check-in. Zero passenger action.

An AI-powered pipeline that proactively checks in every eligible, opted-in passenger — documents verified, seat assigned, boarding pass delivered. The trigger window is fully configurable: from 15 days before departure (early doc validation) to 48h (final check-in). The passenger does nothing.

Proactive, not reactive
The pipeline doesn’t wait for the passenger. Up to 15 days before the flight it validates identity documents and flags expiration issues. At 48h, it executes full automated check-in. No more “set an alarm for check-in” moments.
Smart eligibility — automate only when safe
Not every passenger can be auto-checked in. The pipeline verifies profile completeness, document status, ticketing, and special service requests before proceeding. If any gate fails, it falls back gracefully with a clear, actionable message.
AI-powered document verification
Right after booking, the passenger uploads their identity document. Gemini 2.0 Flash reads passports, DNIs, and ID cards from any country. Images never stored — privacy by design.
Under 15 seconds, check-in pipeline end-to-end
8 orchestrated stages complete in <15 seconds per passenger. Configurable per airline and route. Adapts to domestic, EU, and international flows across IAG.
The Big Vision → Decision Engine
Invisible Check-In is the first module of a broader AI platform we call the Decision Engine — an orchestration layer that anticipates, decides, and acts before the customer even knows there is something to do.

⚠
We know the edge cases — pilot scoped tight
Seat cannibalization: default random seat + “upgrade before check-in” prompt. Coercion risk: transparent rules, editable results. Doc/visa exceptions: graceful fallback. IRROPS: automation pauses. Pilot scope: single passenger, simple domestic trip, no restrictions — hardest cases later.
